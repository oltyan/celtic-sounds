# Celtic Sounds — Max for Live MIDI Instrument

A Max for Live MIDI Instrument device that plays 8 Celtic instruments (Tin Whistle, Irish Flute, Uilleann Pipes, Anglo Concertina, Accordion, Highland Bagpipe, Smallpipes, Säckpipa) with support for WARBL wind controller and keyboard input. The device combines sampled Celtic instruments with real-time expression control and customizable drone accompaniment for authentic and expressive playback.
